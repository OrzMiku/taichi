# ☯ Taichi

A Minecraft performance optimization modpack that supports Fabric and NeoForge mod loaders.

## Philosophy

- **Vanilla-Fidelity:** We never add any mods that alter Minecraft's core game mechanics. The modpack is designed to optimize and enhance the vanilla experience, not to create new gameplay.
- **Performance-First:** Beyond performance-optimizing mods, any added aesthetic or Quality of Life (QoL) mods must be guaranteed to have no negative impact on performance, striving to be performance-neutral or even beneficial.
- **Non-Intrusive by Default:** All non-essential features that might alter player habits or the user interface's look-and-feel are disabled by default. These are provided as optional enhancements that players can enable themselves in the settings according to their own preferences.
